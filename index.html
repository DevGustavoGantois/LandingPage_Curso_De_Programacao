Linguagens de programação que serão mostradas no curso:

    • O Python é uma linguagem de programação
    amplamente usada em aplicações da Web
    , desenvolvimento de software,
    ciência de dados e machine learning (ML).

    • C# é uma linguagem de programação
    orientada a objetos e orientada a componentes.

    • C++ é uma das linguagens
    mais versáteis que existem,
    permitindo desenvolver
    desde tarefas simples como
    aplicações na linha de comando ou web,
    até sistemas complexos
    de tempo real, muito usadas
    no mercado financeiro.

    • Java é uma linguagem
    multiplataforma, orientada a objetos e
    centrada em rede que pode ser usada
    como uma plataforma em si. É uma linguagem de
    programação rápida, segura e confiável
    para codificar tudo, desde aplicações
    móveis e software empresarial
    até aplicações de big data
    e tecnologias do servidor.

    • O JavaScript
    surgiu como uma tecnologia
    do lado do navegador para tornar
    as aplicações Web mais dinâmicas.
    Ao usar JavaScript, os navegadores passaram
    a ser capazes de responder a interações do usuário
    e alterar o layout
    do conteúdo na página.
